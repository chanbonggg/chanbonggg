## About Me

Python, PostgreSQL, Next.js 기반으로 데이터 중심 웹 서비스와 퀀트 투자 시스템을 개발합니다.

- Backend API, 데이터 파이프라인, 백테스트 엔진, 관리자 도구 구현에 관심이 있습니다.
- 복잡한 데이터를 수집, 정제, 분석하고 실제 서비스 화면까지 연결하는 작업을 좋아합니다.
- 실험용 코드보다 실행 가능하고 유지보수 가능한 결과물을 만드는 데 집중합니다.
